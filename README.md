# NanoNinoChatBot
Creating a Narrow Chatbot for Nano-Nino, that answers FAQ regarding the company and ignores chats reggarding irrelevant questions. basic react frontend for frontend, and nest js for creating api, that feeds the chat, the database of the comapany details is stored in postgree SQL, databse contains FAQs, Documentation, Policies, Product info.
stack:
-React.js - frontend
-Nest.js - Backend
-Database - PostgreeSql, Chatgpt.api integration
